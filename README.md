# DRS - F1赛事应用

## 项目简介

DRS是一个基于React Native开发的F1赛事浏览应用，为F1赛车爱好者提供实时赛事信息、车手积分、车队排名等功能。应用采用现代化的UI设计，支持多语言，并提供流畅的用户体验。

## 技术栈

- React Native
- Expo
- TypeScript

## API

- [jopica-f1](https://github.com/jolpica/jolpica-f1)
- [f1-api](https://github.com/Rafacv23/F1-api)

## 主要功能

- 自1950年以来的所有F1赛事数据
- 车手积分排名
- 车队积分排名
- 车手个人信息
- 车队个人信息
- 大奖赛赛程
- 比赛结果统计

## 开发指南

### 环境要求

- Node.js lts
- bun
- iOS模拟器（用于iOS开发）
- Android模拟器（用于Android开发）

### 安装依赖

```bash
bun install
```

### 启动开发服务器-ios

```bash
bun expo run:ios
```
